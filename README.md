# Apollo Micro Systems – Premier Explosives Acquisition Valuation

## Overview

This project independently values Premier Explosives Ltd. using public financial statements and compares the intrinsic valuation with Apollo Micro Systems' announced acquisition.

## Objectives

- Perform Financial Statement Analysis
- Build a Discounted Cash Flow (DCF) Model
- Conduct Comparable Company Analysis
- Estimate Enterprise and Equity Value
- Assess acquisition pricing

## Tools

- Microsoft Excel
- Screener.in
- Financial Modeling

## Valuation Methods

- Discounted Cash Flow (DCF)
- EV/EBITDA Trading Comparables

## Key Outputs

- Enterprise Value
- Equity Value
- Investment Recommendation
- Acquisition Deal Memo
